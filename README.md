# 🎵 Spotify UI Clone

A responsive **Spotify-inspired music player UI** built using **HTML5 and CSS3**.  
This project replicates the layout and visual design of Spotify’s web application, focusing on frontend structure, styling, and modern UI principles.

---

## 📌 Project Overview

This project is a static frontend clone of Spotify’s web interface. It demonstrates how to build a clean, structured, and visually appealing music streaming UI using only HTML and CSS.

The application includes:

- 🎧 Sidebar navigation (Home, Search, Your Library)
- 📂 Playlist / Library section
- 🎵 Album / Music cards with images
- ▶️ Bottom music player control bar
- 🌙 Modern dark-themed UI

> ⚠️ Note: This is a UI-only project. It does not include backend functionality or real music playback.
